# Sourcetree 설치

⚡️ 목표

버전 관리 프로그램 Sourcetree를 설치합니다.

Sourcetree는 Jira로 유명한 Atlassian에서 만든 프로그램으로 오로지 버전 관리에 특화된 무료 애플리케이션입니다. CLI는 어렵고, 에디터에 내장된 기능이 부족하다면 전용 프로그램이 도움이 됩니다.

# macOS

  1. 공식 홈페이지 (opens new window) 접속

  1. Download for Mac OS X 버튼 선택

  1. 다운받은 파일을 압축 해제하고 Sourcetree 아이콘을 드래그하여 Applications로 이동

  1. 애플리케이션을 실행하고 Continue 선택

  1. Git에서 사용자 정보를 설정했다면 추가 설정 없이 Done 선택

  1. 설치 완료! 🎉

# Windows

  1. 공식 홈페이지 (opens new window) 접속

  1. Download for Windows 버튼 선택

  1. Bitbucket 선택 후 Next 버튼 선택 -> 로그인 창 (브라우저)

  1. 웹페이지에서 원하는 방식을 선택하여 로그인 후 애플리케이션으로 돌아옴 왜 윈도우만 강제 로그인이지? ㅠㅠ

  1. 로그인 아이디 확인 후 Next 선택

  1. Mercurial은 사용하지 않기 때문에 체크 해제 후 Next 선택

  1. 사용자 정보 설정 후 Next 선택

  1. 설치 완료 🎉
