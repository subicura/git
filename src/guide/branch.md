# 작업 분기

::: tip ⚡️ 목표
동시에 여러 작업을 진행하고 합치고 충돌을 해결하는 방법을 알아봅니다.
:::

## git switch -c - 브랜치 생성

```
git switch -c
```

## git switch - 브랜치 이동

```
git switch
```

## git merge - 브랜치 합치기

```
git merge
```

## conflict - 충돌 해결

```
git
```
