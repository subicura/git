# 기본 명령어

::: tip ⚡️ 목표
저장소 생성부터 기본적인 히스토리 조작 방법을 알아봅니다.
:::

## git init - 저장소 생성

```
git init
```

## git status - 현재 상태보기

```
git status
```

## git add - 현재 상태 추적

```
git add
```

## git commit - 현재 상태 저장

```
git commit
```

## git reset - 이전 상태로 (이력 제거)

```
git reset
```

## git revert - 이전 상태로 (이력 유지)

```
git revert
```
