# GitHub 공유

::: tip ⚡️ 목표
GitHub 원격 저장소와 연동 후 GitHub Flow를 이용하여 소스를 수정합니다.
:::

## git push - 원격 저장소 저장

```
git push
```

## git clone - 머지 저장소 복제

```
git clone
```

## git pull - 원격 저장소 내용 가져오기

```
git pull
```

## GitHub Flow

```
git rebase
```
