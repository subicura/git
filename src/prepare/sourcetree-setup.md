# SourceTree 설치

::: tip ⚡️ 목표
SourceTree를 설치합니다.
:::

![SourceTree](./imgs/intro/sourcetree.png)

소스트리는 오로지 버전 관리에 특화된 애플리케이션입니다. CLI는 어렵고, 에디터에 내장된 기능이 부족하다면 전용 프로그램이 도움이 됩니다. 써보고 마음에 든다면 다른 유료 프로그램도 알아보세요.

버전 관리에 특화된 애플리케이션 중 대표적인 무료 프로그램을 설치합니다.

## macOS

1. [공식 홈페이지](https://www.sourcetreeapp.com/) 접속

![sourcetree homepage](./imgs/sourcetree-setup/sourcetree-osx.png)

2. `Download for Mac OS X` 버튼 선택

<div style="text-align: center; margin-top: 15px">
  <img src="./imgs/sourcetree-setup/sourcetree-osx-2.png" alt="SourceTree Install" style="width: 500px; max-width: 100%" />
</div>

3. 다운받은 파일을 압축 해제하고 `Sourcetree` 아이콘을 드래그하여 `Applications`로 이동

![sourcetree install](./imgs/sourcetree-setup/sourcetree-install-1.png)

4. 애플리케이션을 실행하고 `Continue` 선택

![sourcetree install](./imgs/sourcetree-setup/sourcetree-install-2.png)

5. Git에서 user 정보를 설정했다면 추가 설정없이 `Done` 선택

<div style="text-align: center; margin-top: 15px">
  <img src="./imgs/sourcetree-setup/sourcetree-install-3.png" alt="SourceTree Install" style="width: 550px; max-width: 100%" />
</div>

7. 설치 완료! 🎉

## Windows

1. [공식 홈페이지](https://www.sourcetreeapp.com/) 접속

![sourcetree homepage](./imgs/sourcetree-setup/sourcetree-windows.png)

2. `Download for Windows` 버튼 선택

<div style="text-align: center; margin-top: 15px">
  <img src="./imgs/sourcetree-setup/sourcetree-install-win-1.png" alt="SourceTree Install" style="width: 500px; max-width: 100%" />
</div>

3. `Bitbucket` 선택 후 `Next` 버튼 선택 -> 로그인 창 (브라우저)

![sourcetree login](./imgs/sourcetree-setup/bitbucket-login.png)

4. 웹페이지에서 원하는 방식을 선택하여 로그인 후 애플리케이션으로 돌아옴 ~~왜 윈도우만 강제 로그인이지? ㅠㅠ~~

<div style="text-align: center; margin-top: 15px">
  <img src="./imgs/sourcetree-setup/sourcetree-install-win-2.png" alt="SourceTree Install" style="width: 500px; max-width: 100%" />
</div>

5. 로그인 아이디 확인 후 `Next` 선택

<div style="text-align: center; margin-top: 15px">
  <img src="./imgs/sourcetree-setup/sourcetree-install-win-3.png" alt="SourceTree Install" style="width: 500px; max-width: 100%" />
</div>

6. `Mercurial`은 사용하지 않기 때문에 체크 해제 후 `Next` 선택

<div style="text-align: center; margin-top: 15px">
  <img src="./imgs/sourcetree-setup/sourcetree-install-win-4.png" alt="SourceTree Install" style="width: 500px; max-width: 100%" />
</div>

7. 사용자 정보 설정 후 `Next` 선택

<div style="text-align: center; margin-top: 15px">
  <img src="./imgs/sourcetree-setup/sourcetree-install-win-complete.png" alt="SourceTree Install" style="width: 500px; max-width: 100%" />
</div>

8. 설치 완료 🎉
